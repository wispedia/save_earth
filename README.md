## 拯救地球

- 一个种树的微信小游戏，将所有树苗种在地球上即认为成功，树苗之间相互碰撞则认为失败
- 基于[cocos creator](https://www.cocos.com/)构建
- 逻辑代码在 `assets/scripts` 文件夹下
